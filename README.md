\# Vulnerability Tracker



A full-stack vulnerability tracking and reporting tool for penetration testing engagements. Built as a portfolio project to demonstrate full-stack development skills and offensive security domain knowledge.



\## Stack

\- \*\*Backend:\*\* Python, FastAPI, PostgreSQL

\- \*\*Frontend:\*\* React

\- \*\*Lab Environment:\*\* DVWA, Metasploitable 2, Kali Linux (VirtualBox)



\## Status

🚧 In active development — Stage 1: Environment Setup complete.



\## Features (Planned)

\- REST API for managing engagements, hosts, and findings

\- CVSS-based severity classification

\- React dashboard with severity breakdowns and filtering

\- PDF/Markdown report generation

\- Real findings documented from a controlled home lab environment
